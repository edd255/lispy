The interpreter is written in C and is based on \href{https://buildyourownlisp.com/}{\texttt{ "{}Build Your Own Lisp"{}}} by Daniel Holden and uses his \href{https://github.com/orangeduck/mpc}{\texttt{ mpc}} library.\hypertarget{md_README_autotoc_md1}{}\doxysection{\texorpdfstring{Modifications}{Modifications}}\label{md_README_autotoc_md1}

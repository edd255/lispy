\doxysubsection{Detailed Description}
A structure that contains data and metadata about used environments. 

The env\+\_\+t struct encodes a list of relationships between symbols (names) and values. This is implemented by using two lists of equal length, where each entry in one list has a corresponding entry in the other list at the same position.
